# Learning Management System (LMS)

This project is a simple and effective Learning Management System (LMS) designed to improve the efficiency of teaching and learning activities.  
The system supports multiple user roles, each with different responsibilities and access levels:

- Admin  
- Instructor  
- Course Creator  
- Student  

The main goal of this LMS is to centralize course-related activities, reduce manual work, and make it easier for institutions to manage courses and learners.

---

## Features

### Admin
- Manages all user accounts.  
- Assigns roles and permissions.  
- Monitors platform activity.  
- Ensures smooth functioning of the LMS.

### Instructor
- Manages courses assigned to them.  
- Uploads learning materials.  
- Creates and manages assessments.  
- Tracks student progress and performance.

### Course Creator
- Creates and designs new courses.  
- Defines course structure (modules/units/topics).  
- Prepares content for instructors to deliver.  
- Keeps courses updated in coordination with instructors.

### Student
- Enrolls in available courses.  
- Accesses course materials and assessments.  
- Submits assignments and quizzes.  
- Views grades, feedback, and learning progress.

---

## How This LMS Improves Efficiency

- Reduces manual work related to course and user management.  
- Offers clear separation of responsibilities among roles.  
- Makes course content easily accessible to students.  
- Enhances coordination between Admin, Instructors, and Course Creators.  
- Provides a structured environment for learning and teaching activities.
